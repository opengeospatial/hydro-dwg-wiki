# **<a href="GroundWater" class="wikilink">GroundWater</a> Standards Working Group** (GW SWG)

The GW SWG aims to:

- Develop a data exchange standard for the groundwater domain. The standard is to be known as GroundWater Markup Language 2 (GWML2) and will consist of conceptual and logical information models, as well as language specific implementations, which initially minimally include a GML-XML encoding specification.
- Maintain and update the standard once established.

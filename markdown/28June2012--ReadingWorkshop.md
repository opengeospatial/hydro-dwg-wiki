# **GWIE2 Breakout Session,** **Hydrology DWG Workshop 2012**

This workshop continues the directions outlined in the successful [Delft Workshop](12April2011--DelftWorkshop), in which it was agreed to move to a harmonized international GML schema for groundwater data. Since that [Delft Workshop](12April2011--DelftWorkshop), significant progress has been made by INSPIRE and OGC/WMO on schemas that include groundwater elements, and the GWML1 schema has been extensively tested in N.America. In this workshop we will review these developments and chart a concrete plan toward an OGC/WMO standard.

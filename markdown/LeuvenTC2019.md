## Hydrology Domain Working Group

## Meeting details at Leuven TC, 24 - 27 December, 2019

Venue: KU Leuven, Hexagon, Informatie Vlaanderen, and Merkator, Leuven Belgium

Time: Wednesday 26 June 2019, 4:30 PM - 6:00 PM \[10:30 AM - 12:00 PM (EDT - US/Eastern)\]

Link to [Leuven TC meeting details](http://ogcmeet.org/#about)

### **Meeting Agenda and Outcomes**

1\. Quick report on status of each each <a href="WaterML2" class="wikilink">WaterML2</a> standard: 5-10 min + short discussion.

–Will submit some change requests to <a href="HY_Features" class="wikilink">HY_Features</a> based on experience using the spec.

–GWML2 is on track to WMO adoption this winter.

\<span style="background-color: transparent;"\>2. Report on WMO submission process + next steps.\</span\>

\<a href="<https://external.opengeospatial.org/twiki_public/pub/HydrologyDWG/LeuvenTC2019/OGC_TC_Leuven_HDWG.pptx>" target="\_blank"\>See slides for overview.\</a\>

\<span style="background-color: transparent;"\>3. Planning for HDWG group annual meeting in Florence.\</span\>

–Will hold remote mixed participation sessions instead of a face to face annual meeting.

\<span style="background-color: transparent;"\>4. Discussion of potential HDWG future work.\</span\>

–Need to continue work on publishing feature models as simple OWL ontologies.

–Interest in pushing forward on shared environmental domain query building blocks for OGC API

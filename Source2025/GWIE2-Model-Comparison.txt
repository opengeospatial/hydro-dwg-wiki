---+ *Groundwater Information Model Comparison*
---++ Models
   * <p> *Groundwater ML1:* [[http://ngwd-bdnes.cits.rncan.gc.ca/gwml/index.aspx?c=2][documentation]] and [[http://www.iwaponline.com/jh/014/jh0140093.htm][paper]]</p>
   * *INSPIRE hydrogeology model:*
      * *Latest version (3.0rc2) accepted by TWG Geology and publicly available*
         * Documentation as a part of Data Specification on Geology
            * PDF: http://inspire.jrc.ec.europa.eu/documents/Data_Specifications/INSPIRE_DataSpecification_GE_v3.0.pdf
            * HTML version of UML model: http://inspire.ec.europa.eu/data-model/approved/r4618/html/
         * UML model
            * Inspire web site data models section (go here to always check you have the latest version) : http://inspire.ec.europa.eu/index.cfm/pageid/2/list/datamodels
            * XMI in INSPIRE repository: https://inspire-twg.jrc.ec.europa.eu/svn/inspire-model/branches/public/approved/latest/themes/annexII/geology/IR_HydrogeologyCore.xml
            * EA and XMI zipped:http://inspire.ec.europa.eu/data-model/approved/r4618/ea+xmi/EAXMI.zip
            * <span style="background-color: transparent;">XSD in INSPIRE repository: </span><span style="background-color: transparent;">http://inspire.ec.europa.eu/schemas/ge_hg/3.0/HydrogeologyCore.xsd </span>
      * *Old full version (0.4) rejected by TWG Geology due to it is too complex and presented here as a internal document of GWIE2*
         * Automatically generated documentation of UML model: http://testbed.ptip.org.pl/GWIE2_INSPIRE/INSPIRE_Hydrogeology_Draft_Outline_v.0.4_Doc.docx
         * EA version of UML model: http://testbed.ptip.org.pl/GWIE2_INSPIRE/INSPIRE_Hydrogeology_Draft_Outline_v.0.4.EAP
         * The model also contains Use_Case diagrams worked out by Piotr Wojda. Description of the Use_Case: http://testbed.ptip.org.pl/GWIE2_INSPIRE/UseCase_HydrogeologicalModel_UC.doc
         * More hydrogeological Use_Cases you can find in INSPIRE Data Specification – Geology (UC06, UC07, UC08, UC09 and UC 10 - pages from 123 to 134) at: http://inspire.jrc.ec.europa.eu/documents/Data_Specifications/INSPIRE_DS_Geology_v3.0_rc2.pdf
   * %BLUE%<u>Hydrogeological Map of Bavaria (from Bernhard Wagner)</u>%ENDCOLOR% : [[%ATTACHURL%/20030246_SystemdokumentationV1.0_AnhangB.pdf][20030246_SystemdokumentationV1.0_AnhangB.pdf]]
   * BRGM / SANDRE
      * Main SANDRE web site http://www.sandre.eaufrance.fr/

      * ADES web services : <a href="http://www.ades.eaufrance.fr/IMG/pdf/BRGM_RP-59919-FR.pdf" title="blocked::http://www.ades.eaufrance.fr/IMG/pdf/BRGM_RP-59919-FR.pdf">http://www.ades.eaufrance.fr/IMG/pdf/BRGM_RP-59919-FR.pdf</a>
